Exercise 1

This exercise examines the differences between using the linear regression model versus the generalized linear regression model where the distribution of the response variable is considered normal. You will find the data in the file practical2_lowbwt.sav.

Perform a linear regression of the variable BWT (birth weight) on the explanatory variables SMOKE and LWT (weight of the mother during last menstrual period).

For R Users!

use lm(...) for the model and confint() for the confidence intervals. If you have two models, m1 and m2, you may compare them by using modelsummary(list(m1, m2), stars=TRUE)

Compare these results with the results of an analysis in GzLM (SPSS users) or glm(…) (R users) in which you define the distribution of the response variable as normal (let the software create confidence intervals for the regression coefficients). Do the results agree? Where do they differ and why?
